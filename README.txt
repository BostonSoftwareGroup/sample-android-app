This is an Android Application. You can simply run this by open it in Android Studio and press the green arrow button on the top bar.

This application has following features:
	1. In main screen: 
		a. Users can choose to see all airlines' list or favorite airlines' list by swiping left or right.
		b. Clicking airline item in list can lead users to airline detail screen.
	2. In airline detail screen:
		a. Users can visit airline website and make calls to the airline company by clicking url and phone number.
		b. Users can add or remove current airline in favorite list by clicking on float button on right bottom corner.
